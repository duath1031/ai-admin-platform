# Admini 4.0 Post-문서24 통합 개발계획

## 핵심 비전
- "문서24는 시작일 뿐, 기업의 심장(돈, 사람, 땅)을 다룬다"
- 최종 목표: 대한민국 모든 기업의 AI COO (J.A.R.V.I.S)

## Phase 구조

### Phase 0: 핵심 기반 (Foundation)
- SSO Vault: 카카오/네이버(완료), 계정 금고 AES-256(진행중), 나라장터/홈택스/고용산재토탈/하이코리아/관세청(대기)
- 결제: 토스페이먼츠(1회+구독), 카카오페이
- AI 서류 생성: HWPX(진행중), DOCX(완료), PDF(완료)

### Phase 1: 민원 행정 (Module A) - 현재
- 3-Way 하이브리드: ①직접접수(무료) ②문서24봇(9,900원/월) ③전문가대행(50,000원/건)
- AI 기능: 만능검색, 서식보관함, 민원길잡이, 서류검증, 음성민원, 규제모니터링, 기한매니저

### Phase 2: 조달/입찰 & 자금 (Module B) - 다음 즉시
- **무제한 입찰 시뮬레이터** (SaaS 핵심, 49,000원/월)
  - 나라장터 공고 파싱 → 사정율 분포 분석 → 슬라이더로 투찰가 조정
  - 특허 회피: AI "예측/추천" X, 과거 데이터 "분석 도구" O
- 스마트 필터: 면허/실적/신용 기반 1순위 가능 공고 필터링
- AI 프로젝트 매니저: 입찰마감/보증서/제안서/기성 알림 (카톡)
- 보증보험 서류 자동 생성 (입찰/이행/계약/하자)
- 벤더/컨소시엄 매칭 (공동도급, 하도급)
- 정책자금/지원사업 매칭 + 기업인증 자동화

### Phase 3: 기업 운영 & 인사 (Module C)
- R&D 연구노트 & 전자결재 (Killer Feature) - 블록체인 타임스탬프
- 노무/4대보험 자동화 (입사/퇴사/급여) - 50인 미만 사업장 타겟
- 외국인 채용/비자 (E-7, E-9, D-10) - 하이코리아 연동
- 법인차량 운행기록부 자동화

### Phase 4: 안전/환경 (Module D)
- 중대재해: TBM 음성인식 자동화, 위험성평가 AI 이미지분석
- 환경: 올바로시스템 연동, 폐기물 신고

### Phase 5: 부동산/개발 (Module E)
- 토지 분석 리포트 (LURIS + 조례 + 대장)
- 세움터 건축행정 연동

### Phase 6: 법무/계약 (Module F)
- AI 계약서 리스크 분석 (독소조항 감지)
- 내용증명/지급명령 자동 작성
